AI-powered note-taking application with rich text editing. Built with Vue 3 and TipTap for the editor, Supabase for backend storage and authentication.

Features include real-time collaboration, AI-assisted writing suggestions, and markdown export. The application supports multiple notebooks with tagging and full-text search.
