An note-taking application that combines a familiar editing interface with the power of AI models. The project is built using the Feature-Sliced Design (FSD) architecture, ensuring high scalability and maintainable code.

Key features:
- **AI-Powered Editing**: Generate, modify, and refine text directly within the editor using AI models via Groq.
- **Privacy and Reliability**: All notes and settings are securely stored locally in your browser (IndexedDB).
- **Advanced Organization**: Keep thoughts structured with tags and set reminders for your notes.
- **Modern Tech Stack**: Built with Vue 3 and Tiptap, leveraging Supabase Edge Functions for backend logic and an automated CI/CD pipeline.
