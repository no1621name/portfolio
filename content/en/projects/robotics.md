This project features a 3D-printed robotic hand model capable of mimicking operator gestures in real-time with high precision. By combining computer vision powered by Mediapipe with socket-based communication to Trik controllers, the system achieves seamless human-machine interaction. The engineering rigor behind this solution has been recognized with multiple awards at regional school project competitions.
