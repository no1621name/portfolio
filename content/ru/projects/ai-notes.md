Приложение для заметок с ИИ-помощником и редактированием обогащённого текста. Построено на Vue 3 и TipTap для редактора, Supabase для хранения и аутентификации.

Возможности включают совместную работу в реальном времени, предложения по написанию с помощью ИИ и экспорт в markdown. Приложение поддерживает несколько блокнотов с тегами и полнотекстовым поиском.
