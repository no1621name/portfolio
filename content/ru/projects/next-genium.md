Корпоративный сайт для перспективной геймдев-студии Next Genium, создающей игры для ПК и консолей.

Разработал проект с нуля, написав кастомную WordPress-тему по предоставленному дизайну. Для реализации сложных и нестандартных интерфейсных решений по собственной инициативе интегрировал Web Components, что позволило сделать UI более интерактивным, модульным и удобным для поддержки.
