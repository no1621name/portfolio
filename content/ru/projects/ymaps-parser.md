Приложение для парсинга отзывов об организациях с Яндекс Карт. Принимает ссылку на организацию, извлекает базовую информацию (название, рейтинг, количество отзывов) и собирает все доступные отзывы с пагинацией.

Архитектура: SPA на Vue 3 общается с Laravel-бэкендом через REST API с Sanctum аутентификацией. Тяжёлая операция парсинга вынесена в очередь, прогресс транслируется на фронт через Server-Sent Events (SSE).

Основные особенности:
- **Двухуровневая стратегия парсинга** — основной API-парсер с параллельными запросами (concurrency = 3) и автоматический fallback на headless-браузер (Puppeteer через spatie/browsershot) при ошибках API
- **Обработка rate-limit** — при получении 429 автоматически переключается на последовательные запросы с рандомной задержкой, при повторном 429 — одна попытка повтора
- **Canary-проверка** — ежедневная автоматическая проверка работоспособности API Яндекса через artisan-команду, выполняет полный pipeline без записи в БД
- **Хранение отзывов** — отзывы сохраняются в собственной БД для расширяемости (добавление новых метаданных без потери старых) и удобства бизнес-аналитики

Фронтенд построен по архитектуре Feature-Sliced Design (FSD). В качестве UI-библиотеки используется DaisyUI поверх Tailwind CSS, для HTTP-запросов — ofetch. Сетевой путь парсинга покрыт тестами на фикстурах (10 модульных, 3 интеграционных, тесты headless-парсера).
